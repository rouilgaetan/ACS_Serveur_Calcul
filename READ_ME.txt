Projet Architecture Client/Serveur :

Pour lancer le projet vous avez besoin de la bibliothèque gson normalement présente dans l'archive rendu.

Le client est le projet ClientCalcul.

Le serveur est le projet web ServeurCalcul.

Il y a une page web de test incluse dans le serveur des tests plus rapide.

Les opérateurs définis pour ce projet sont :

-plus : unaire et binaire
-minus : unaire et binaire
-sqrt : unaire
-inv : unaire

-mult: binaire
-div : binaire

Tous les opérateurs sont à écrire en argument tels qu'ils sont écrit ici (sans tenir compte de la casse).

Pour le type du résultat du calcul le serveur renvoie un double si au moins un double est présent dans les opérandes.
Sinon le serveur renvoie un integer.

Exceptions gérées :

-Division par zéro
-Mauvais format de nombre
-Opérateur inexistant
-Pas d'argument
-Argument vide

Merci d'avoir lu, et bon courage pour la suite des corrections.
